# Sistema de Voto Electrónico

Proyecto integrador desarrollado para las asignaturas de Ingeniería de Software, Modelamiento de Software y Herramientas de Programación.

## Descripción
Sistema de voto electrónico que permite la autenticación de usuarios, gestión de elecciones, emisión de votos de forma segura y consulta de resultados, garantizando el voto único, anónimo y auditable.

## Tecnologías Utilizadas
- ASP.NET Core Web API
- PostgreSQL
- Entity Framework Core
- JWT (JSON Web Token)
- Swagger
- GitHub

## Roles del Sistema
- ADMIN: Gestión de elecciones y candidatos.
- VOTANTE: Emisión de voto electrónico.

## Funcionalidades Principales
- Autenticación y autorización por roles
- Creación y gestión de procesos electorales
- Registro de voto único por usuario
- Consulta de resultados
- Exportación de reportes

## Ejecución del Proyecto
1. Clonar el repositorio
2. Configurar la cadena de conexión en `appsettings.Development.json`
3. Ejecutar migraciones con Entity Framework
4. Ejecutar el proyecto desde Visual Studio
